# med-rag
 this is a medical rag solution that based on medical data trained LLm 

 ## Requirments:
 Based on python 3.10 or later

- Create Conda environment :
 ```bash
$ conda create -n med-rag-env python=3.10
```
- Activate the environment:
```bash
$ conda activate med-rag-env
```

#### For more suitable and readability the bash terminal use:
```bash
export PS1="\[\033[01;32m\]\u@\h:\w\n\[\033[00m\]\$ "
```

### For installing all required packages:
``` bash
$ pip install requirments.txt
```
### For setup the environmental variables:
 ```bash
 $ cp .env.example .env
 ```
 Set your environment variables in the `.env` file. Like `OPENAI_API_KEY` value.
